# 🏛️ Civic Complaint Management System



---

## 📖 Project Overview

The Civic Complaint Management System is a full-stack web application that digitalizes the entire civic complaint management process. The platform enables citizens to report civic issues by submitting complaints with relevant details and supporting evidence. Admin review and verify incoming complaints before assigning them to the appropriate workers for resolution. Assigned workers update the progress of their tasks and mark complaints as resolved upon completion. Throughout the process, citizens can monitor the status of their complaints in real time, ensuring transparency and efficient communication between citizens, administrators, and field workers.

### 🔹 Project Summary

- 🏛️ Digital platform for managing civic complaints
- 👥 Supports three user roles: Citizen, Administrator, and Worker
- 📋 Allows citizens to submit complaints online
- 📤 Enables administrators to verify and assign complaints
- 👷 Allows workers to update progress and resolve complaints
- 🔄 Provides real-time complaint tracking
- 🔐 Secure role-based authentication
- 📷 Supports image uploads for complaint evidence
- 📊 Centralized complaint management dashboard
- ✅ Improves transparency, accountability, and service efficiency

## ✨ Key Features

### 👤 Citizen

- Register and log in securely
- Submit civic complaints
- Upload supporting images
- Provide complaint details
- Track complaint progress
- View complaint history
- Manage user profile

### 👨‍💼 Admin

- Log in to the admin panel
- Review submitted complaints
- Verify complaint information
- Assign complaints to workers
- Manage users
- Monitor complaint progress
- Update complaint status
- Access dashboard overview

### 👷 Worker

- Log in to the worker panel
- View assigned complaints
- Update work progress
- Mark complaints as resolved
- View complaint details
